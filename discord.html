Retro Bot

RetroBot was created by n2va and has a variety of commands. The bot plays music, suggests some of the best synthwave playlists and wallpapers, and has multiple fun commands

Example Commands:

  • !play LINK - plays a song from a youtube link

  • !skip - skips the current song

  • !price [BTC, ETH, LTC] - returns the current price of the specified cryptocurrency

  • !wp - returns a random outrun themed wallpaper
